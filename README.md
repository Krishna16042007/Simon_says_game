# 🎮 Simon Says Game

A fun and interactive memory-based Simon Says game built using **HTML, CSS, and JavaScript**. The game challenges players to remember and repeat an increasingly long sequence of colors. As the levels progress, the sequence becomes more complex, testing memory, concentration, and reaction skills.

## 🚀 Features

* Interactive gameplay
* Random color sequence generation
* Increasing difficulty with each level
* Score and level tracking
* Instant feedback for correct and incorrect moves
* Simple and responsive user interface
* Keyboard-based game start

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📸 Preview

Players start the game by pressing any key. A sequence of colors is displayed, and the player must repeat the sequence correctly. Each successful round adds a new color to the pattern, making the game more challenging.

## ⚙️ How to Run

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Open the project folder.

3. Run `index.html` in your browser.
